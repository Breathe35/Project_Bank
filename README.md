# Bank Credit Repayment Analysis

Анализ данных банка для выявления факторов, влияющих на своевременное погашение кредита.  
Проект выполнен в рамках учебного портфолио.

---

## Описание

Заказчик — кредитный отдел банка. Необходимо проверить, влияют ли **семейное положение** и **количество детей** на факт погашения кредита в срок.  
Результаты могут быть использованы для построения модели кредитного скоринга.

---

## Данные

Исходные данные — статистика о платёжеспособности клиентов (файл `data.csv`).  
Содержит 21 471 запись (после удаления дубликатов) и 12 полей:

- `children` — количество детей  
- `days_employed` — трудовой стаж (в днях)  
- `dob_years` — возраст (в годах)  
- `education` — уровень образования  
- `education_id` — идентификатор образования  
- `family_status` — семейное положение  
- `family_status_id` — идентификатор статуса  
- `gender` — пол  
- `income_type` — тип дохода  
- `debt` — факт невозврата (0/1)  
- `total_income` — ежемесячный доход  
- `purpose` — цель кредита  

---

## Задачи проекта

1. Изучить данные, выявить пропуски и аномалии.
2. Очистить данные:  
   - заменить пропуски в доходе медианными значениями по группам;  
   - исправить ошибки в возрасте, стаже, количестве детей;  
   - привести текстовые поля к единому формату.
3. Провести лемматизацию целей кредита для группировки.
4. Проанализировать влияние на невозврат кредита:  
   - количества детей;  
   - семейного положения;  
   - уровня дохода;  
   - цели кредита.
5. Сформулировать выводы и рекомендации для кредитного отдела.
